# Cost of Living Dashboard

## Setup

1. Install dependencies:

   ```bash
   npm install
   ```

2. (Optional) Re-process data:

   ```bash
   npm run process-data
   ```

   _Note: Requires Node.js environment._

3. Run locally:
   ```bash
   npm run dev
   ```

## Tech Stack

- React (Vite)
- D3.js
- TopoJSON
